Web System: Invoice Management Sistema Web: Gestão de Notas Fiscais

PythonStreamlitPandas

A full-stack application developed to replace manual spreadsheets in financial control. The system ensures standardization, security, and data validation for Invoices.

Uma aplicação full-stack desenvolvida para substituir planilhas manuais no controle financeiro. O sistema garante a padronização, segurança e validação das informações de Notas Fiscais.

  • User Authentication with session control.
  • Full CRUD Operations (Create, Read, Update, Delete).
  • Validation Dictionary: prevents typos by locking inputs to approved suppliers.
  • Advanced Filters: Paginated search to handle large data volumes smoothly without freezing the UI.
  • Autenticação de Usuários com controle de sessão.
  • Operações CRUD completas (Cadastro, Edição, Exclusão e Leitura).
  • Dicionário de Validação: impede erros de digitação travando inputs em fornecedores homologados.
  • Filtros Avançados: Busca paginada para lidar com grandes volumes de dados sem travar a interface.
